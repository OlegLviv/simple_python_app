Simple Python App
This is a simple Python app that demonstrates some basic features of the language. It can:

Print “Hello, world!” to the console
Perform arithmetic operations such as addition, subtraction, multiplication and division
Generate a random number between 1 and 100
Convert Celsius to Fahrenheit and vice versa
Check if a word is a palindrome or not
Requirements
To run this app, you need:

Python 3.6 or higher
A code editor of your choice
Usage
To run this app, open a terminal and navigate to the folder where the app.py file is located. Then type:

python app.py
Copy
You will see a menu with different options. Enter the number of the option you want to choose and press Enter. Follow the instructions on the screen.

To exit the app, enter 0 and press Enter.
